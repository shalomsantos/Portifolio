## 💼 O Projeto: BookFlix

Este portfólio simula uma plataforma de gerenciamento de biblioteca inteligente. O foco foi criar uma experiência fluida de ponta a ponta.

### O que este projeto demonstra sobre mim:
* **Atenção à Experiência do Usuário (UX):** Interface limpa e transições fluidas usando Vuetify para que o usuário não se sinta perdido.
* **Preocupação com Segurança:** Integração com arquitetura Stateless (JWT) no Spring Boot, protegendo rotas no Front-end com middlewares eficientes.
* **Capacidade de Entrega:** Um ecossistema completo (Full Stack) que conversa perfeitamente entre cliente e servidor.

---

## 🛠️ Tecnologias que utilizo para entregar valor
* **Front-end:** Nuxt 3, Vue 3, Vuetify
* **Back-end:** Java, Spring Boot (Security, MVC)
* **Estado & Dados:** Nuxt State Management (`useState`/`useCookie`), Axios/Fetch
